#Markdown语法简介  
1. 标题的使用
-一级标题>#
-二级标题>##
-三级标题>###
2. 段落与换行
-段落:直接写文字,段落间用空行隔开
-换行:两个空格,再换行
4. 强调
-**加粗**(用`**文字**`或者`__文字__`)
-*斜体*(用`*文字*`或者`_文字_`)
-***斜体加粗***(`***`)
-~~删除线~~(`用~~文字~~`)
5. 无序列表
- `-,+,*`加空格
5. 有序列表
- `1. `
6. 链接
  ```[显示的文字](https://链接地址)```
7. 图片
  ```![替代文本](图片链接地址)```
8. 引用
  >在段落前加`>`
9. 代码
  -行内代码`i`
  -整块代码```i```
10. 分割线:单独使用一行`---,***,___`
11. 表格
  -使用竖线`|`和`-`绘制表格
  ```
  |表头一 |表头二|
  |------|------|
  |单元格1|单元格2|
