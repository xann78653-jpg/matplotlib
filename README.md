# Markdown语法简介  
1. 标题的使用
  -一级标题`# `
  -二级标题`## `
  -三级标题`### `
3. 段落与换行
  -段落:直接写文字,段落间用空行隔开
  -换行:两个空格,再换行
4. 强调
-**加粗**(用`**文字**`或者`__文字__`)
-*斜体*(用`*文字*`或者`_文字_`)
-***斜体加粗***(`***`)
-~~删除线~~(`用~~文字~~`)
5. 无序列表
 - `-,+,*`加空格
6. 有序列表
 - `1. `
7. 链接
  ```[显示的文字](https://链接地址)```
8. 图片
  ```![替代文本](图片链接地址)```
9. 引用
  >在段落前加`>`
10. 代码
  -行内代码`i`
  -整块代码
```i```
11. 分割线:单独使用一行`---,***,___`
12. 表格
  -使用竖线`|`和`-`绘制表格
  ```
  |表头一 |表头二|
  |------|------|
  |单元格1|单元格2|
